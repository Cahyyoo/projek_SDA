Selamat Datang di Menu Help Quebyx
Quebyx adalah platform simulasi berbasis web untuk mempelajari konsep queue. Berikut panduan menggunakan game ini:

Start:
Halaman awal game. Klik "Mulai" untuk masuk ke menu utama.

Menu Utama:
New Game: Memulai simulasi queue. Anda akan diarahkan ke halaman FIFO, LIFO, Priority Order (PO), dan SIRO secara berurutan.
Help: Menampilkan panduan penggunaan (halaman ini).
Quit: Menutup game.

Simulasi Queue:
Di setiap halaman (FIFO, LIFO, dll.), gunakan tombol yang tersedia untuk memanipulasi elemen antrian.
Pelajari bagaimana elemen diproses sesuai jenis antrian.
